# Changelog

All notable changes to Guileless Bopomofo are documented here, split by major version.

- [4.x](CHANGELOG-4.x.md) — User Phrase Manager, redesigned settings, security audit
- [3.x](CHANGELOG-3.x.md) — Android 16, Material Design themes, physical keyboard overhaul, DaChen CP26
- [2.x](CHANGELOG-2.x.md) — Rust-based libchewing, redesigned keycaps, conversion engine options
- [1.x](CHANGELOG-1.x.md) — Initial release through Material Design 3, physical keyboard support, dark theme
- [0.x](CHANGELOG-0.x.md) — Pre-release development: project bootstrap, libchewing integration, keyboard UI
